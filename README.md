# 🎬 Video Play - Просмотр сериалов

Полнофункциональное React приложение для просмотра сериалов с поддержкой нескольких источников видео (VK Video и Rutube).

## ✨ Возможности

- ✅ 8 сезонов с разным количеством серий
- ✅ Просмотр видео с двумя источниками (VK Video и Rutube)
- ✅ Отслеживание просмотренных серий
- ✅ Сохранение прогресса в localStorage
- ✅ Адаптивный дизайн для всех устройств
- ✅ Темная тема по умолчанию
- ✅ Индикатор загрузки видео
- ✅ Прогресс просмотра сезона

## 🛠 Технологический стек

- **React 18** - JavaScript библиотека для создания пользовательских интерфейсов
- **Vite** - современный инструмент для быстрой разработки
- **Tailwind CSS** - utility-first CSS фреймворк
- **JavaScript ES6+** - современный JavaScript

## 🚀 Запуск проекта

### Установка зависимостей

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Приложение откроется на `http://localhost:5173`

### Сборка для production

```bash
npm run build
```

### Просмотр build версии

```bash
npm run preview
```

## 💡 Хранение данных

Приложение использует `localStorage` для сохранения:

- Последний выбранный сезон (`lastSeason`)
- Последний открытый эпизод (`lastEpisode`)
- Список просмотренных серий (`watchedEpisodes`)

## 🎬 Структура данных

### Сезон

```javascript
{
  seasonNumber: 1,
  episodeCount: 10,
  episodes: [...]
}
```

### Эпизод

```javascript
{
  id: "S1E1",
  title: "Серия 1",
  description: "Сезон 1, серия 1",
  vkUrl: "https://vk.com/video_ext.php?...",
  rutubeUrl: "https://rutube.ru/play/embed/...",
  isWatched: false
}
```

## 🎨 Темная тема

Приложение использует темную тему с цветовой палитрой:

- **Основной цвет:** `#111827` (очень темный серый)
- **Вторичный цвет:** `#1f2937` (темный серый)
- **Акцентный цвет:** `#2563eb` (голубой)
- **Текст:** `#f3f4f6` (светло-серый)

## 🔧 Компоненты

### Header

Заголовок приложения с названием и описанием

### SeasonSelector

Кнопки для выбора сезона с отображением количества серий

### EpisodeList

Сетка карточек с серийми

- Клик открывает плеер
- Отмечает просмотренные серии зеленой галочкой
- Показывает выделение для выбранной серии

### VideoPlayer

Модальное окно с плеером

- Встраивание видео через iframe
- Переключение между источниками
- Индикатор загрузки
- Автоматическая отметка как просмотренной

### SourceSwitcher

Кнопки для переключения между VK Video и Rutube

## 📱 Адаптивность

Приложение полностью адаптивно:

- **Мобильные устройства:** 1 колонка
- **Планшеты:** 2-3 колонки
- **Десктопы:** 4 колонки

## 🐛 Обработка ошибок

- Проверка доступности видеоссылок
- Отображение сообщений об ошибках в плеере
- Fallback на альтернативный источник при недоступности
